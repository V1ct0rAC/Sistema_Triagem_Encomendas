🎯 Desafio de Código: O Sistema de Triagem de Encomendas

Cenário: Você recebeu uma lista bruta contendo os códigos de rastreio de vários pacotes. O sistema
antigo gerou erros: alguns códigos estão duplicados e eles vieram completamente bagunçados.

Sua missão é criar uma classe em Java que processe esses dados removendo duplicidades, ordenando
alfabeticamente e associando cada código ao nome do cliente de destino.
